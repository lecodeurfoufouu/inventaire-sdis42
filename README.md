# 🚒 Inventaire SDIS 42 — CIS Séverine

Fiches d'inventaire des engins et application mobile de check de garde.

## Liens

- **Affiche (ordinateur)** : https://lecodeurfoufouu.github.io/inventaire-sdis42
- **Check de garde (mobile)** : https://lecodeurfoufouu.github.io/inventaire-sdis42/check.html

## Fichiers

| Fichier | Description |
|---------|-------------|
| `index.html` | Affiche A3 imprimable avec flèches et QR codes |
| `check.html` | Application mobile de prise de garde |

## Mise à jour

Pour mettre à jour l'inventaire :
1. Ouvrir `index.html` sur GitHub
2. Cliquer l'icône ✏️ (Edit)
3. Modifier les données
4. Cliquer "Commit changes"

Le site se met à jour automatiquement en ~1 minute.
